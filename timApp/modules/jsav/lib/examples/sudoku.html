JSAV Sudoku Grid

This is a simple example of the JSAV matrix. It also illustrated updating slideshows interactively, on-the-fly. The example creates a sudoku game using a JSAV matrix. If you click on cells, they highlight the first time you click. Each click adds a new slide, which you can see by resetting the slideshow with the << control. View the source to see how this is done in CSS and JavaScript.
